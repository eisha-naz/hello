----------------------------------------
Basic Git Workflow
----------------------------------------

1️⃣ Clone a Repository  
git clone <repo_link>  
✔️ Downloads the repository to your local machine.  

2️⃣ List Files  
ls  
✔️ Shows the files in the current directory.  

3️⃣ Change Directoryg
cd <folder_name>  
✔️ Moves into the specified folder.  
✔️ Any new files or edits will be made inside this folder.  

----------------------------------------
File Management
----------------------------------------

🔹 Create a New File (PowerShell Alternative to touch)  
New-Item hello.html -ItemType File  
✔️ Creates an empty file named hello.html.  
to move it for github/ to hello/ folder
Move-Item test2.html hello/

🔹 Verify File Creation  
ls  
✔️ Lists files to check if hello.html was created.  

🔹 What is a Directory?  
✔️ A folder that stores files or other folders.  

----------------------------------------
Using VS Code
----------------------------------------

🔹 Open Current Directory in VS Code  
code .  
✔️ Opens the current folder in Visual Studio Code.  

----------------------------------------
Git Commands Explained
----------------------------------------

1️⃣ git add → Selecting Pages to Save  
🔹 Marks changes to be saved (does not save yet).   Tll git what file to keep track of
🔹 Like putting items in your cart before checkout.  
git add <file_name>  

2️⃣ git commit → Taking a Snapshot  
🔹 Saves the current state of tracked files.   
🔹 Allows you to go back to this version later.  
git commit -m "Your commit message"  

3️⃣ git push → Uploading to GitHub  
🔹 Sends your committed changes to GitHub.  
🔹 Even if your computer is lost, files remain online.  
git push origin main  

----------------------------------------
Extra Git Commands
----------------------------------------

🔹 Unstage a File (Remove from git add)  
git reset <file_name>  

🔹 See Commit History  
git log  

🔹 Go Back to an Old Version  
git checkout <commit_id>

🔹 This .git folder stores all version history and tracking data.
🔹 It turns your folder into a Git repository so Git can track changes.

🚀 BASIC GIT WORKFLOW

1️⃣ Clone a Repository  
   $ git clone <repo_link>

2️⃣ List Files  
   $ ls

3️⃣ Move into a Folder  
   $ cd <folder_name>

📂 FILE MANAGEMENT

🔹 Create a File  
   $ New-Item hello.html -ItemType File

🔹 Move a File  
   $ Move-Item test.html hello/

🔹 Check If a File Exists  
   $ Test-Path hello.html

🖥️ OPEN IN VS CODE

🔹 Open Folder  
   $ code .

🔹 Open a File  
   $ code filename

🚀 GIT COMMANDS

1️⃣ Stage Files  
   $ git add <file_name>

2️⃣ Commit Changes  
   $ git commit -m "Your message"

3️⃣ Push to GitHub  
   $ git push origin main

🔄 EXTRA COMMANDS

🔹 Check Git Status  
   $ git status

🔹 Unstage a File  
   $ git reset <file_name>

🔹 View Commit History  
   $ git log

🔹 Go to an Old Version  
   $ git checkout <commit_id>

🛠️ FIRST-TIME SETUP

🔹 Initialize a Git Repo  
   $ git init

🔹 First-Time Push  
   $ git push -u origin master
